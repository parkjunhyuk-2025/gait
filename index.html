ZMP 개념과 인간형 로봇의 균형 제어

인간의 보행 안정성 지표(XcoM·MoS)와 인간형 로봇의 영점모멘트점(ZMP) 제어는 같은 문제 — "지지기저면 안에서 무게중심을 어떻게 유지할 것인가" — 를 서로 다른 방식으로 풀어낸 결과입니다.

지지기저면(BOS)과 안정성 여유

개념도
무게중심 COM 확장무게중심 XcoM 안정성 여유 MoS

계산 원리

수식
ω₀ = √(g / l) — 도립진자 고유진동수, l = 다리 길이
XcoM = COM + vCOM / ω₀ — Hof et al., 2005
MoS = min(XcoM − BOSmin, BOSmax − XcoM)
MoS > 0 이면 XcoM이 지지기저면 안에 있어 안정, MoS < 0 이면 경계를 벗어나 수정 스텝(교정 발디딤)이 필요한 상태로 해석합니다. 로봇 제어에서는 같은 논리를 ZMP가 지지다각형 경계를 벗어나지 않도록 하는 문제로 치환해 풉니다 (Vukobratović & Borovac, 2004).
